# Predicting-ozone-concentration-in-Kaohsiung-using-weather-factors
此專案為我高中時期做過的科展內容，利用天氣因子透過類神經網路去預測高雄地區的臭氧濃度，但由於當時在指導老師的奇怪建議下選擇投了數學組，而導致只拿了佳作。

## 前置作業
### 資料蒐集
大部分的資料取自高雄市氣象局，包含高雄全部的行政區。

### 特徵選擇
使用皮爾森相關係數搭配多項式回歸的反向淘汰法來了解哪些特徵是比較重要的。

### 模型建立
其實當時我和我的組員都缺乏一定的程式能力且指導老師也非此領域的專業，所以在找到資料後選擇上網找模板，因此在只知道些許名詞的情況下就做了這個專案，使用了多項式回歸、決策樹、隨機森林和刀時正夯的類神經網路模型。

##  研究結果

### 特徵
透過皮爾森相關係數搭配多項式回歸的反向淘汰法有成功找到對模型有幫助的特徵，且也有實際的化學反應式得以佐證。

### 模型結果
最後的 R-square 是0.66，算是一個普通的擬和度，也為了迎合科展數學的成分畫了很多擬和圖和方程式，

## 心得
這一次科展其實是承去年某一組的研究進行延伸的，當時我以能為校代表出賽為榮，但當最後拿到佳作獎狀時，我理解這不過是那位指導老師想壓榨這份研究的最後一份價值，畢竟對外行人而言佳作仍是一個獎項，但對圈內人而言，高雄市科展數學組的佳作就只是鼓勵你有做事情跑完流程的參加獎。但不可否認的是，這次科展正是我想投入資料科學領域的啟蒙專題，也鼓勵我後續願意投入複雜麻煩的珊瑚礁專案。雖然那次比賽是一個笑話，但仍對我意義重大。
